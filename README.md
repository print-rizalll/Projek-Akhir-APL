# HMS : Hotel Management System untuk Hotel Grans Rizuna Hotel

Program ini adalah sistem manajemen reservasi hotel berbasis konsol (CLI) yang dikembangkan menggunakan bahasa C++. Sistem ini memungkinkan dua jenis pengguna admin dan user biasa untuk mengelola pemesanan kamar hotel secara terstruktur. Admin memiliki akses penuh terhadap seluruh data reservasi, termasuk membuat, melihat, mengubah, menghapus reservasi, mencari data tamu, mengurutkan daftar kamar, serta menambahkan akun admin baru; sementara user biasa hanya dapat melakukan reservasi dan melihat riwayat pemesanan miliknya sendiri. Setiap reservasi mencatat data tamu secara lengkap, mulai dari nama, NIK (Nomor Induk Kependudukan 16 digit), nomor HP, hingga tanggal check-in, sehingga beberapa tamu dapat memesan kamar yang sama di hari yang berbeda. Program dilengkapi dengan validasi input menggunakan regex (format NIK, HP, tanggal, username), penanganan error (exception handling dan validasi cin), algoritma pengurutan (selection sort, bubble sort, insertion sort), serta binary search rekursif untuk pencarian kamar.
